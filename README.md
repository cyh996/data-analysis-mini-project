# Data Analysis Mini Projects

This repository contains a collection of mini data analysis projects completed using Python, pandas, and matplotlib.

The projects focus on data cleaning, exploratory data analysis (EDA), aggregation, and data visualization techniques commonly used in real-world data analysis workflows.

## Projects

### Daily Sales Trend Analysis

* Sales trend analysis using time-series data
* Data aggregation and visualization
* Business insight extraction

### Online Store Sales Analysis

* Product and sales performance analysis
* Revenue and customer behavior exploration
* Visualization of key business metrics

### Student Score Analysis

* Student performance analysis
* Subject-wise score comparison
* Statistical summary and visualization

## Tech Stack

* Python
* pandas
* NumPy
* matplotlib

## Purpose

This repository was created to practice fundamental data analysis skills and build a strong foundation for larger-scale data analysis and machine learning projects.
